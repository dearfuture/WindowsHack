第九章	病毒,又见病毒

1.案例说明：
Virus为本章案例对应的解决方案;
Virus.sln为解决方案文件,使用Visual Studio 2008打开;
Release文件夹下的Virus.exe为案例对应可执行文件;
案例用于演示病毒相关原理及实现方法。

2.推荐测试环境：
操作系统：	Windows XP sp3
虚拟机：	VMWare 6.5 + Windows XP sp3
编译器：	Visual Studio 2008
编译选项：	字符集设置为未设置
build版本：	release版本

3.使用说明：
测试时请务必关闭系统防护软件,不推荐在实体机内进行测试;
请关闭系统还原功能,否则系统会自动修复被感染文件导致测试不成功;
该案例具有危害性，仅供学习参考,请勿随意散播。
