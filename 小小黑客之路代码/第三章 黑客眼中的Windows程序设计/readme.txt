第三章	黑客眼中的Windows程序设计

1.案例说明：
WinProgramming为本章案例对应的解决方案;
WinProgramming.sln为包含了所有项目的解决方案文件,使用Visual Studio 2008打开;
Release文件夹下的UseAPI.exe、TestDll.exe为案例对应的可执行文件;
TestDll.exe需要和CustomDll.dll放置于同一文件夹下。

案例用于演示Windows编程中API、DLL的使用方法:
UseAPI是一个演示MessageBox()使用的项目;
CustomDll是一个完整的DLL文件代码;
TestDll、TestDll2两个项目分别演示了两种调用Dll代码的方法。

2.推荐测试环境：
操作系统：	Windows XP sp3
编译器：	Visual Studio 2008
编译选项：	字符集设置为未设置
build版本：	release版本

3.使用说明：
无。
