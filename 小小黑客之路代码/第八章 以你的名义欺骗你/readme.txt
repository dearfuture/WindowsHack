第八章	以你的名义欺骗你

1.案例说明：
ArpAttack为本章案例对应的解决方案;
ArpAttack.sln为解决方案文件,使用Visual Studio 2008打开;
Release文件夹下的ArpAttack.exe为案例对应可执行文件;
案例用于演示ARP攻击的原理及实现方法。

2.推荐测试环境：
操作系统：	Windows XP sp3
虚拟机：	VMWare 6.5 + Windows XP sp3
编译器：	Visual Studio 2008
编译选项：	默认设置
build版本：	release版本

3.使用说明：
测试时请务必关闭系统防护软件;
推荐使用VMware搭建两台虚拟机协同实体机进行测试，两台虚拟机和一台实体机必须在VMware搭建的同一局域网内且相互可Ping通;
因为VMware的设置原因,请勿使用实体机作为欺骗或被欺骗端。
