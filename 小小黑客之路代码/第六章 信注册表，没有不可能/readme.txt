第六章	信注册表,没有不可能

1.案例说明：
RegEdit为本章案例对应的解决方案;
RegEdit.sln为解决方案文件,使用Visual Studio 2008打开;
Release文件夹下的RegEdit.exe为案例对应可执行文件;
案例为注册表编辑器,可以用于枚举、查找、删除、编辑各注册表键。

2.推荐测试环境：
操作系统：	Windows XP sp3
编译器：	Visual Studio 2008
编译选项：	默认设置
build版本：	release版本

3.使用说明：
个别子键由于帐号权限可能会出现打开失败的情况;
新建或修改键值时请勿随意输入过长的键值名称或数据;
请勿随意删除系统的自建或键值、以免导致系统出错。
